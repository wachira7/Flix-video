# FlixVideo Mobile

AI-Powered Movie & TV Show Discovery Mobile App

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- Yarn
- Expo CLI
- iOS Simulator (Mac only) or Android Emulator

### Installation

```bash
# Install dependencies
yarn install

# Start Expo development server
yarn start

# Run on iOS
yarn ios

# Run on Android
yarn android
```
